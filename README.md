# Multimodal Parkinson’s Disease Detection

This repository contains the code for a multimodal Parkinson’s disease detection framework using voice and gait data.

## Modalities
- Voice: UCI Parkinson’s Dataset (ID 174)
- Gait: PhysioNet Gait in Parkinson’s Disease Dataset

## Method
- Modality-specific encoders
- Attention-based multimodal fusion
- Robustness via modality dropout

## Usage
Datasets are not included in this repository. Please download them from their respective sources.
